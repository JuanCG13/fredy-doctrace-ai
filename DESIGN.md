# DocTrace AI Design System

DocTrace AI uses the exact Google Stitch HTML exports as the visual source for the public Vercel demo. The implementation adds only progressive interaction JavaScript for safe portfolio actions.

## Visual direction
- Dark enterprise document-intelligence cockpit.
- Navy surfaces, cyan primary actions, emerald indexed/grounded states, amber risk/open-question states, violet AI/report accents.
- Dense internal-tool layout with sidebar, topbar, document processing cards, report builder, source evidence panel and processing logs.

## Safety
All content is fictional. No private documents, credentials, compliance claims or external sends are used.
